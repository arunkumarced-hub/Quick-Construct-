Quick Construct – Dual Service Web Platform

Quick Construct is a web platform designed for the construction and hardware industry.

The website has two main services:

Hiring skilled and semi-skilled workers such as plumbers, electricians, carpenters, painters, welders, AC technicians, and general helpers.

Buying construction materials and hardware tools from nearby hardware vendors through an online system.

The project includes separate frontend interfaces for:

Admin – manages users, orders, vendors, and workers

Vendor – lists products, manages orders, and billing

Worker – receives job requests and manages availability

Client – hires workers and purchases tools/materials

This repository currently contains the completed frontend structure of all modules.
The next phase will include backend integration, database connection, and deployment.

Tech Used:
HTML, CSS, JavaScript (Frontend)
